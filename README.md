Heron - Sistema Web para Estetica y Barberia

Sistema web full-stack desarrollado para Heron Estetica y Barberia en Durango, Mexico. Permite a los clientes conocer los servicios, reservar y pagar en linea, y obtener su factura electronica. Incluye gestion de inventario y control de transacciones en base de datos.

CARACTERISTICAS PRINCIPALES

Catalogo de servicios: listado con precios, descripcion y servicios destacados
Reserva y pago en linea: carrito de servicios con checkout integrado mediante Conekta
Webhooks de pago: actualizacion automatica del estado de transacciones en tiempo real
Facturacion electronica: generacion de CFDI mediante la API de Facturama
Autenticacion de usuarios: registro, login y cierre de sesion con manejo seguro de cookies
Control de inventario: administracion de productos y servicios desde base de datos
Diseno responsivo: compatible con dispositivos moviles y escritorio

TECNOLOGIAS UTILIZADAS

JavaScript: logica del frontend e interactividad
PHP: backend y logica del servidor
CSS y Bootstrap: diseno y estilos responsivos
MySQL: base de datos relacional
Conekta API: procesamiento de pagos en linea
Facturama API: facturacion electronica CFDI

SEGURIDAD IMPLEMENTADA

Cookies de sesion con HttpOnly y SameSite Lax
Rutas protegidas con guard.php
Consultas con prepared statements para prevenir SQL Injection
Variables de configuracion separadas de la logica de negocio

APRENDIZAJES CLAVE

Desarrollo full-stack sin frameworks, PHP puro y JavaScript vanilla
Integracion de pagos reales y manejo de webhooks en produccion
Generacion de facturas electronicas con estandares SAT Mexico
Arquitectura limpia separando API, configuracion, utilidades y vistas
